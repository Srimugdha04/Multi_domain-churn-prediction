Churn Prediction Dashboard Docs

This documentation explains how to use the Customer Churn Prediction Dashboard, including data upload, analytics, prediction logic, and insights generation.

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v14 or higher)
  • npm or yarn package manager
  • Modern browser (Chrome recommended)
  • Basic knowledge of CSV data format
  • Internet connection (for charts rendering)

Installation

  1. Clone the repository or download the template
  2. Navigate to the project directory
  3. Install dependencies: 
    npm install

Key Features

  • 📊 Dynamic churn analytics dashboard
  • 📁 CSV upload and processing
  • 📉 Churn vs Retained visualization
  • 🔍 Customer-level churn prediction
  • 🧠 Explainable AI (reasons + recommendations)
  • 📄 Data explorer with filtering & pagination

How to Use

  1. Go to Dashboard page
  2. Upload CSV dataset
  3. View churn analytics and charts
  4. Navigate to "Churn Dashboard" to explore customers
  5. Use "Predict Customer" to analyze individual users
npm run dev

Prediction Logic

The system uses a rule-based scoring model to estimate churn risk. 

                    
                    Credit Score < 500 → High risk
                    Low Balance → Increased risk
                    Low Salary → Moderate risk
                    Inactive Member → High risk
                    High Complaints → Very high risk

Next Steps

  1. Review the main entry point in src/js/main.js
  2. Customize components according to your needs
  3. Build for production: 
    npm run build
